# SFD2IR
This repository is the offical implementation for "Multiscale Spatial-Frequency Learning for  Degradation Decoupling in RS Image Restoration."[[IEEE TMM](https://ieeexplore.ieee.org/document/11353927)] 

## Datasets
Cloud Removal:[(https://github.com/haidong-Ding/Cloud-Removal)]  
Deblurring:[(https://captain-whu.github.io/BED4RS)]   
Dehazing:[(https://www.dropbox.com/s/k2i3p7puuwl2g59)] and [(https://github.com/IDKiro/DehazeFormer)]    
Super-Resolution:[(https://github.com/wjwjww/OLI2MSI)]
